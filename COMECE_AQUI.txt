Bem-vindo ao Sistema ERP Lince Track!

Este é um sistema completo de gestão para empresas de rastreamento veicular,
desenvolvido com Next.js 14, TypeScript, Tailwind CSS e Supabase.

🚀 INÍCIO RÁPIDO (5 MINUTOS)
================================================================================

1. Instalar dependências:
   npm install

2. Copiar variáveis de ambiente:
   cp .env.example .env.local

3. Editar .env.local com suas credenciais do Supabase

4. Executar schema SQL no Supabase (arquivo: supabase/schema.sql)

5. Criar primeiro usuário admin no Supabase

6. Rodar o projeto:
   npm run dev

7. Acessar: http://localhost:3000

Para instruções detalhadas, consulte CHECKLIST_INSTALACAO.md

================================================================================
